# ポケカ対戦タイマー PWA v2

25分1試合用のポケカ対戦タイマーです。iPhoneの回転ロックがONでも使えるように、縦向き画面の中身だけを90度回転させる「疑似横向き表示」に対応しています。

## 使い方

1. GitHub PagesなどHTTPSで公開します。
2. iPhoneのSafariで公開URLを開きます。
3. 共有メニューから「ホーム画面に追加」します。
4. ホーム画面のアイコンから起動します。
5. 回転ロックONのまま、iPhoneを横向きに持って使います。

画面が上下逆に感じる場合は、中央下部の「向き反転」を押してください。向きの設定は端末内に保存されます。

## ファイル構成

- `index.html` : 画面本体
- `styles.css` : レイアウトと疑似横向き表示
- `app.js` : タイマー処理
- `manifest.webmanifest` : PWA設定
- `sw.js` : オフラインキャッシュ
- `icons/` : ホーム画面用アイコン
