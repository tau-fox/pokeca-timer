# ポケカ対戦タイマー PWA v3

v3では以下を調整しました。

- 回転ロックON向けの疑似横向き表示を維持
- メインタイマーを縮小してはみ出しにくく調整
- 余白を減らして横幅を有効活用
- 左側パネルの要素を右に90度、右側パネルの要素を左に90度回転
- プレイヤーA/B表記を削除
- 対戦開始を押した側に「先攻」、押さなかった側に「後攻」を表示
- 左右入替ボタンを削除
- Service Workerキャッシュを v3 に更新

更新手順:

1. GitHubリポジトリの既存ファイルをこのv3の中身で上書きする
2. 反映を1〜3分待つ
3. iPhoneのSafariで `https://tau-fox.github.io/pokeca-timer/index.html?v=3` を一度開く
4. その後ホーム画面アイコンから起動する
