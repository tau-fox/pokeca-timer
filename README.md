# ポケカ対戦タイマー PWA v4

v4では以下を調整しました。

- プレイヤーエリアの回転レイアウトを再構成し、ボタン・タイマー・先攻後攻の重なりを解消
- プレイヤーエリア内の各要素を専用スロットに分離
- 画面全体を少し下寄せして、時計とダイナミックアイランドとの重なりを軽減
- 余白が出る場合はプレイヤーエリアの内容を下寄せ
- Service Worker キャッシュを v4 に更新

更新手順:

1. GitHubリポジトリの既存ファイルをこのv4の中身で上書きする
2. 反映を1〜3分待つ
3. iPhoneのSafariで `https://tau-fox.github.io/pokeca-timer/index.html?v=4` を一度開く
4. その後ホーム画面アイコンから起動する
