# ポケカ対戦タイマー PWA v1

## 概要

iPhoneを横向きにして使う、ポケカ対戦用の25分タイマーです。
持ち時間制ではなく、中央のメインタイマーが25分から減ります。
左右のプレイヤーごとに、そのプレイヤーがターン中に使った時間を累積表示します。

## 使い方

1. `index.html` をHTTPSで配信できる場所に置く、またはローカルサーバーで開きます。
2. iPhoneのSafariで開きます。
3. 共有メニューから「ホーム画面に追加」を選びます。
4. ホーム画面のアイコンから起動します。
5. iPhoneを横向きにして使います。

## ボタン挙動

- 初期状態では左右どちらの「対戦開始」でも開始できます。
- 押した側が現在ターン側になり、そのボタンは「ターン終了」に変わります。
- 反対側の開始ボタンは非活性になり、ラベルは非表示になります。
- 一時停止ボタンは左右に表示され、試合中のみ押せます。
- 一時停止すると全タイマーが止まり、現在ターン側の「ターン終了」が「試合再開」に変わります。
- 試合再開を押すと、同じプレイヤーのターンから再開します。
- 中央の「リセット」で初期状態に戻します。

## ファイル構成

- `index.html` : 画面本体
- `styles.css` : 横向きiPhone向けUI
- `app.js` : タイマー処理とボタン挙動
- `manifest.webmanifest` : PWAインストール設定
- `sw.js` : オフライン利用用Service Worker
- `icons/` : PWA用アイコン
